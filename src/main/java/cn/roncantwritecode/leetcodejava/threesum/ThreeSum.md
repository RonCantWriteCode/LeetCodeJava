### 题目描述
11. 三数之和

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例1:

```
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
```

示例2:
```
输入：nums = []
输出：[]
```

示例3:
```
输入：nums = [0]
输出：[]
```

提示：

* 0 <= nums.length <= 3000
* -105 <= nums[i] <= 105



### 题解

#### 算法1  双指针 O(n*n)

结果要求三元组不重复，所以第一步应该对数组进行排序。三数之和为0，循环迭代数组，确定第一个数字，剩余问题即对于有序的数组求两个数字 b+c=-a，可以使用双指针。由于双指针时间复杂度O(n),所以时间复杂度为O(n*n)。
